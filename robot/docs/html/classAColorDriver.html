Description détaillée

Interface abstraite du driver de capteur couleur.

Fournit la lecture des coordonnees chromatiques (Tx, Ty) depuis un capteur couleur I2C. L'implementation ARM lit le capteur reel, il n'y a pas d'implementation SIMU (ARM seulement).
