Description détaillée

Interface abstraite du driver de servomoteurs.

Fournit le controle des servomoteurs : positionnement par impulsion PWM, mode moteur continu, gestion du couple. L'implementation ARM pilote les servos reels (Dynamixel AX-12, PWM Adafruit), la SIMU est un stub.
