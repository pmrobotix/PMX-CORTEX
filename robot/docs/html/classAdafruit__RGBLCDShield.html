Description détaillée

Driver pour le LCD Shield RGB 16x2 Adafruit.

Singleton. Communique avec le LCD et les boutons via le MCP23017 I2C. Fournit l'affichage texte, le controle du backlight et la lecture des 5 boutons du shield.
