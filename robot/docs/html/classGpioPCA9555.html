Description détaillée

Driver singleton pour le GPIO expander PCA9555 via I2C.

Port0 = sorties (setOnP0/setOffP0), Port1 = entrees (getValueP1). Utilise pour la tirette et les switchs arriere sur l'OPOS6UL.
