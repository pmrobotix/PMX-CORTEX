Description détaillée

Implementation ARM du driver de LEDs.

Implementation simulation du driver de LEDs.

Chaque LED est controlee par un GPIO de l'OPOS6UL. Les numeros GPIO sont calcules selon la formule Armadeus : GPIO_num = (Bank - 1) * 32 + Pin.

Stocke l'etat des LEDs dans un tableau et affiche les changements dans la console via le logger.
