Description détaillée

Implementation ARM du driver de switchs.

Implementation simulation du driver de switchs.

Utilise le PCA9555 (port1 = entrees) pour lire la tirette (pin 7) et les switchs arriere (pins 0-1). Le port0 sert de sorties GPIO.
