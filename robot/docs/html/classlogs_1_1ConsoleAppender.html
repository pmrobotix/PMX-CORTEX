Description détaillée

Implémentation de Appender pour une écriture des traces sur un flux de sortie de type std::ostream.

Cette implémentation se base sur un thread secondaire pour l'écriture dans le fichier des informations.

Le constructeur par défaut de cette classe permet de créer un Appender vers la sortie standard.
