Description détaillée

Structure pour gérer une boucle périodique sans dérive.

Utilise clock_nanosleep avec TIMER_ABSTIME pour cibler des instants absolus. Exemple d'utilisation :

PeriodicTimer timer(100000); // période 100ms en microsecondes
while (running) {
do_work();
timer.sleep_until_next();
}
