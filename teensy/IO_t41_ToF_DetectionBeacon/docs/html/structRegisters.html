Detailed Description

Registres I2C en lecture seule pour le master (OPOS6UL).

Contiennent les resultats de detection : positions (x, y, angle), distances, donnees brutes par zone et flags de status.

Note
Les registres sont mis a jour a chaque cycle d'acquisition et proteges par un mutex (registers_new_data_lock).
